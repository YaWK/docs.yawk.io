Detailed Description

Throws a fancy Bootstrap Alert (success, info, warning or danger)

Throw default bootstrap-powered alerts. (warning, succes etc...) This lets you throw an alert message box at any time, anywhere in your script. Alert uses 5 Arguments, as shown below. Example: any place of your script where an error, info, success or danger Message needs to be thrown.

Class covers both, backend & frontend functionality. See Methods Summary for Details!
